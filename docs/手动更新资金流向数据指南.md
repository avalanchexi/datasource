# 手动更新资金流向数据指南

**目的**: 当MCP WebSearch不可用时，手动更新报告中的资金流向数据

**工具**: `scripts/utility/manual_fund_flow_updater.py`

---

## 步骤1: 获取实时数据

### 1.1 北向资金 (Northbound)

**数据源**: [东方财富网-沪深港通](https://data.eastmoney.com/hsgt/)

**需要获取的数据**:
- 近5日净流入金额（亿元）
- 近120日累计净流入金额（亿元）
- 流向趋势（持续流入/流出/震荡）

**示例命令**:
```bash
python scripts/utility/manual_fund_flow_updater.py \
  --market-data data/20251112_market_data_final_enhanced.json \
  --flow-type northbound \
  --recent-5d "+132.6亿" \
  --total-120d "+845.2亿" \
  --trend "持续流入" \
  --source "东方财富网实时数据" \
  --note "数据日期: 2025-11-12"
```

---

### 1.3 ETF资金流 (ETF)

**数据源**: Wind/Choice终端 或 [东方财富ETF数据中心](https://fund.eastmoney.com/data/fundranking.html)

**需要获取的数据**:
- 近5日ETF净申购金额（亿元）
- 近120日累计净申购金额（亿元）
- 流向趋势

**示例命令**:
```bash

  --market-data data/20251112_market_data_final_enhanced.json \
  --flow-type etf \
  --recent-5d "+45.8亿" \
  --total-120d "+328.5亿" \
  --trend "持续流入" \
  --source "Wind终端/东方财富ETF中心" \
  --note "数据日期: 2025-11-12"
```

---

### 1.4 融资融券 (Margin)

**数据源**:
- [上交所-融资融券](http://www.sse.com.cn/market/dealingdata/overview/margin/)
- [深交所-融资融券](http://www.szse.cn/market/dealingdata/rzrq/margin/)

**需要获取的数据**:
- 近5日融资融券余额变化（亿元）
- 近120日累计余额变化（亿元）
- 流向趋势

**示例命令**:
```bash
python scripts/utility/manual_fund_flow_updater.py \
  --market-data data/20251112_market_data_final_enhanced.json \
  --flow-type margin \
  --recent-5d "+88.5亿" \
  --total-120d "+1248.3亿" \
  --trend "持续增加" \
  --source "上交所+深交所官方数据" \
  --note "数据日期: 2025-11-12, 两市融资融券余额17256.8亿元"
```

## 步骤2: 批量更新脚本

如果您已经收集了所有4类数据，可以创建批量更新脚本：

**创建文件**: `update_fund_flow_batch.bat` (Windows) 或 `update_fund_flow_batch.sh` (Linux/Mac)

```batch
@echo off
REM Windows批量更新脚本

set MARKET_DATA=data\20251112_market_data_final_enhanced.json

REM 北向资金
python scripts\utility\manual_fund_flow_updater.py ^
  --market-data %MARKET_DATA% ^
  --flow-type northbound ^
  --recent-5d "+132.6亿" ^
  --total-120d "+845.2亿" ^
  --trend "持续流入" ^
  --source "东方财富网" ^
  --note "数据日期: 2025-11-12"

REM 南向资金
python scripts\utility\manual_fund_flow_updater.py ^
  --market-data %MARKET_DATA% ^
  --flow-type southbound ^
  --recent-5d "+68.3亿" ^
  --total-120d "+512.7亿" ^
  --trend "震荡流入" ^
  --source "同花顺" ^
  --note "数据日期: 2025-11-12"

REM ETF资金流
python scripts\utility\manual_fund_flow_updater.py ^
  --market-data %MARKET_DATA% ^
  --flow-type etf ^
  --recent-5d "+45.8亿" ^
  --total-120d "+328.5亿" ^
  --trend "持续流入" ^
  --source "Wind终端" ^
  --note "数据日期: 2025-11-12"

REM 融资融券
python scripts\utility\manual_fund_flow_updater.py ^
  --market-data %MARKET_DATA% ^
  --flow-type margin ^
  --recent-5d "+88.5亿" ^
  --total-120d "+1248.3亿" ^
  --trend "持续增加" ^
  --source "上交所+深交所" ^
  --note "数据日期: 2025-11-12"

echo [OK] 资金流向数据更新完成
pause
```

## 步骤3: 重新生成报告

更新market_data.json后，重新生成报告：

```bash
# 方法1: 仅重新生成Stage 3报告
python scripts/stage3_report_generator.py \
  --market-data data/20251112_market_data_final_enhanced.json \
  --pring-result data/20251112_pring_result_final.json \
  --output reports/20251112背景扫描120_已更新.md

# 方法2: 运行完整Stage 4补充（会尝试MCP WebSearch）
python scripts/stage4_mcp_enhancer.py \
  --market-data data/20251112_market_data_final_enhanced.json \
  --pring-result data/20251112_pring_result_final.json \
  --mode supplement \
  --regenerate-report \
  --report-output reports/20251112背景扫描120_Stage4补充.md
```
