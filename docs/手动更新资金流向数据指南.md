# 手动更新资金流向数据指南

**版本**: V4.0
**更新日期**: 2025-12-09
**目的**: 当Stage2自动获取失败时，手动补充资金流向数据

---

## 前置条件

确保已完成Stage1和Stage2基础流程，生成了 `data/${DATE}_market_data_stage2.json`。

---

## 步骤1: 获取实时数据

### 1.1 北向资金 (Northbound)

**数据源**: [东方财富网-沪深港通](https://data.eastmoney.com/hsgt/)

**需要获取的数据**:
- 近5日净流入金额（亿元）- 必须是数字
- 近120日累计净流入金额（亿元）- 必须是数字
- 流向趋势（流入/流出）

### 1.2 南向资金 (Southbound)

**数据源**: [同花顺-港股通](https://data.10jqka.com.cn/hgt/ggta/)

**需要获取的数据**:
- 近5日净流入金额（亿元）
- 近120日累计净流入金额（亿元）
- 流向趋势

### 1.3 ETF资金流 (ETF)

**数据源**: [东方财富ETF数据中心](https://fund.eastmoney.com/data/fundranking.html)

**需要获取的数据**:
- 近5日ETF净申购金额（亿元）
- 近120日累计净申购金额（亿元）
- 流向趋势

### 1.4 融资融券 (Margin)

**数据源**:
- [上交所-融资融券](http://www.sse.com.cn/market/dealingdata/overview/margin/)
- [深交所-融资融券](http://www.szse.cn/market/dealingdata/rzrq/margin/)

**需要获取的数据**:
- 近5日融资融券余额变化（亿元）
- 近120日累计余额变化（亿元）
- 流向趋势

## 步骤3: 运行注入脚本

```bash
python inject_websearch_data_test.py \
  data/${DATE}_market_data_stage2.json \
  reports/websearch_results_${DATE}_manual.json \
  data/${DATE}_market_data_complete.json
```

## 注意事项

1. **数值格式**: `recent_5d`/`total_120d` 必须是数字（如 `132.6`），不能是文字（如 "波动"、"净流入"）
2. **单位统一**: 统一使用亿元
3. **更新后验证**: 注入后检查 `metadata.data_completeness` >= 0.8
4. **参考schema**: 详见 `src/datasource/models/market_data_contract.py`
