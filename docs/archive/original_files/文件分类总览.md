# 项目文件分类总览

本文按功能模块对仓库内文件进行分类整理，便于团队快速定位与维护。

## 1) 核心代码（Core Library）
- 包根目录：`src/datasource/`
  - 管理器与模型
    - `manager.py`：数据源统一管理（主/备源、故障转移、批量接口）。
    - `models/base.py`：配置/请求/响应模型与 `BaseDataSource` 抽象。
    - `__init__.py`：包导出（当前有编码噪声，建议后续修复为纯 UTF-8）。
  - 数据源适配器（Adapters）
    - `adapters/akshare_adapter.py`：AKShare 适配器（限流/缓存/重试/可用性）。
    - `adapters/tushare_adapter.py`：TuShare 适配器（Token、Pro/免费接口、限流/缓存/重试）。
    - `adapters/__init__.py`
  - 工具与缓存（Utils/Cache）
    - `utils/rate_limiter.py`：异步速率限制器与管理器。
    - `utils/retry.py`：异步重试装饰器（指数退避）。
    - `cache/memory_cache.py`：内存缓存（TTL）与 `cache_key` 辅助。
    - `utils/__init__.py`、`cache/__init__.py`
  - 计算与分析（Calculators）
    - `calculators/technical_indicators.py`：MA/斜率/波动率/趋势评分与指数分析。
    - `calculators/bond_calculator.py`：债券收益率（ETF 代理）与收益率曲线分析。
    - `calculators/fund_flow_calculator.py`：北/南向资金估算与聚合。
    - `calculators/pring_analyzer.py`：普林格六阶段判定与建议。
    - `calculators/__init__.py`
  - 引擎与生成器（Engines/Generators）
    - `engines/data_engine.py`：并发拉取与汇总，输出报告可用的结构化数据。
    - `engines/__init__.py`
    - `generators/report_generator.py`：背景扫描/日表报告生成与写文件。
    - `generators/__init__.py`

## 2) 可执行脚本与示例（Scripts / Examples）
- `run_na_filling.py`：命令行入口，生成背景扫描/日表/二者皆有，或检查可用性/演示 N/A 填充。
- `example.py`：统一管理器使用示例（基础/指数/实时/批量/自定义配置/错误处理）。
- `market_scan_data.py`：市场扫描示例（主要指数、实时、市况、ETF 流向占位）。
- 轻量/集成测试
  - `simple_test.py`：轻量连通性测试（简易 logger）。
  - `test_datasource.py`：数据源集成测试（各接口、批量、日志）。
  - `test_na_filling.py`：N/A 填充与报告链路测试（引擎/生成器/计算器/全流程）。

## 3) 配置与安装（Config / Setup）
- `requirements.txt`：依赖清单。
- `setup.py`：包安装元数据（`packages=find_packages(where="src")`）。
- `.env.example`：环境变量示例（`TUSHARE_TOKEN`、限流、缓存配置）。
- `.claude/settings.local.json`：本地工具配置（非运行必需）。

## 4) 模板、文档与说明（Templates / Docs）
- 模板
  - `templates/daily_report_template.md`：日表报告模板（占位符由生成器填充）。
- 文档
  - `README.md`：项目说明、特性、安装、用法、API。
  - `CLAUDE.md`：面向 AI 助手的开发指引与结构说明。
  - `datasource.md`：占位（当前为空）。
  - `docs/报告生成文件关系说明.md`：两类报告与代码/调用链关系说明。
  - **`docs/技术指标计算说明.md`：技术指标计算方法 + 库存周期验证框架（2025.9.10更新）**
  - **`docs/库存周期验证标准.md`：商品趋势双重验证标准与投资决策框架（2025.9.10新增）**

## 6) 产出路径（实际生成的报告）
- 程序生成的报告默认写入：`reports/`
  - 背景扫描：`background_scan_report_<YYYYmmdd_HHMMSS>.md`
  - 日表报告：`daily_table_report_<YYYYmmdd_HHMMSS>.md`
- 注：根目录的 `背景扫描.md`、`日报扫描.md` 为说明/样例，不会被程序自动覆盖。

- `src/datasource/__init__.py` 开头存在编码异常字符，建议转为纯 UTF-8 文本。
- `datasource.md` 为空，可补充模块概览或快速导航。
