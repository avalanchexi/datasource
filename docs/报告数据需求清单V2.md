
#### 计算指标（基于原始数据）

- 近5日涨跌%: `(close[-1] / close[-5] - 1) * 100`
- 近120日涨跌%: `(close[-1] / close[0] - 1) * 100`
- MA50: 50日移动平均
- MA200: 200日移动平均
- MA50斜率: 10期线性回归斜率
- 30日波动率%: `std(returns[-30:]) * sqrt(252) * 100`
- 趋势评分: 四维度评分（收益+均线+趋势+动量）

```python

    阶段 = "被动补库存"  # 价格下降+需求偏弱+供给韧性

    阶段 = "主动补库存"  # 价格上升+需求回暖

    阶段 = "被动去库存"
else:
    阶段 = "主动去库存"
```

**货币宽松度计算**:

```python
货币宽松度 = (
    存准率评分(30分) +  # 降准+30分,不变0分
    逆回购评分(30分) +  # 降息+30分,不变0分
    TSF评分(25分) +     # 增速>10%得25分,5-10%得15分
    M2评分(15分)        # 增速>8%得15分,5-8%得10分
) / 100 * 100
```

## 三、Pring三层框架数据映射

### 3.1 第一层：库存周期分析

**输入数据**: PPI, PMI, 工业增加值, BDI, CPI（见2.6）
**输出结果**:

- 库存周期阶段（被动补库存/主动补库存/被动去库存/主动去库存）
- 商品趋势倾向（偏牛/偏熊/中性）
- 基本面评分（0-60分）

### 3.2 第二层：货币周期叠加

**输入数据**: 降准, 降息, TSF, M2（见2.7）
**输出结果**:

- 货币周期阶段（适度宽松/中性/收紧）
- 权益偏向（支持权益/压制权益）
- 债券偏向（债券优先/权益优先）
- 货币宽松度（0-100分）

### 3.3 第三层：Pring六阶段判定

**输入数据**:

- 债券信号: 基于美国10Y + 中国10Y收益率变化（见2.4）
- 股票信号: 基于沪深300 + 创业板指120日涨跌（见2.1）
- 商品信号: 基于黄金+铜+BCOM的YTD表现（见2.2）

**判定逻辑**:

```python
# 标准Pring六阶段
if 债券↑ and 股票↓ and 商品↓: return "第Ⅰ阶段"
if 债券↑ and 股票↑ and 商品↓: return "第Ⅱ阶段"
if 债券↓ and 股票↑ and 商品↑: return "第Ⅳ阶段"
if 债券↓ and 股票↓ and 商品↓: return "第Ⅵ阶段"
# ... 其他阶段

# 中国市场修正
if 货币宽松度 > 80 and 库存周期 == "被动补库存":
    # 修正到复苏早期
    return "第Ⅰ-Ⅱ阶段过渡期"
```

**输出结果**:

- 最终判定阶段（第Ⅰ-Ⅵ阶段）
- 置信度（0-100%）
- 资产配置建议（债券/股票/商品/现金比例）
