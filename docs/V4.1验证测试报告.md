## 四、关键改进点

### 1. 预分析数据验证

- **V4.0**: 无验证，直接执行分析
- **V4.1**: 三阶段处理，先验证后分析

### 2. 完整性计算

```python
# 第一层完整性
layer1_checks = {
    "PPI数据": macro_data.get('ppi') is not None and len(macro_data.get('ppi', [])) > 0,
    "PMI数据": macro_data.get('pmi') is not None and len(macro_data.get('pmi', [])) > 0,
    "工业增加值": macro_data.get('industrial_value') is not None,
    "CPI数据": macro_data.get('cpi') is not None and len(macro_data.get('cpi', [])) > 0,
}
layer1_completeness = sum(layer1_checks.values()) / len(layer1_checks) * 100
```

### 3. 质量门槛

```python
min_threshold = 60.0  # 最低60%数据完整性
if overall_completeness < min_threshold:
    return {
        "stage": "数据不足",
        "confidence": 0.0,
        "error": f"数据完整性不足 ({overall_completeness:.1f}% < {min_threshold:.0f}%)"
    }
```

### 4. 详细状态报告

- **V4.0**: 无数据质量报告
- **V4.1**: 逐层、逐项详细报告，使用 [OK]/[MISSING] 标记
